Connect to Google Sheets — one-time setup

1
Create a blank Google Sheet. Share it with both parents as Editor. Go to Extensions → Apps Script, paste the apps_script.gs file, click Save.
2
Deploy → New deployment → Web app. Set Execute as: Me, Who has access: Anyone. Deploy, authorize, then copy the Web App URL.
3
Open the Web App URL once in your browser with ?action=initSheets appended — this auto-creates all 6 sheet tabs with default data.
4
Paste the Web App URL below. Do this on both parents' devices — data always syncs through your shared Sheet.
